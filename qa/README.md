# QA — ExitGuard 검증 문서

라하나 대표님(QA) 담당 폴더. 테스트케이스·검증 결과를 코드/기획 문서와 분리해 관리한다.

## 구조

- `test-cases/` — 시나리오별 테스트케이스 (입력·기대결과·엣지케이스)
- `verification/` — 실제 검증 결과 기록 (무엇을 언제 확인했고 결과가 어땠는지)

## 원칙

- MVP 세로 절단 시나리오(김민수 퇴사 → 3레일 → 방어가능 리포트) 기준으로 작성한다.
- 파일명: `YYYY-MM-DD_주제.md` (예: `2026-07-20_ai판례대조-엣지케이스.md`)
